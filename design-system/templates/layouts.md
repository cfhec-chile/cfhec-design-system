# Page Templates

## Dashboard Layout

Structure for main dashboard views with KPIs and activity.

```
┌─────────────────────────────────────────────────────────────────┐
│ [Navbar]                                                        │
├──────────┬──────────────────────────────────────────────────────┤
│          │  Page Header                           [Actions]     │
│          ├──────────────────────────────────────────────────────┤
│          │  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐    │
│ Sidebar  │  │ KPI 1   │ │ KPI 2   │ │ KPI 3   │ │ KPI 4   │    │
│          │  └─────────┘ └─────────┘ └─────────┘ └─────────┘    │
│          ├──────────────────────────────────────────────────────┤
│          │  ┌───────────────────────┐ ┌───────────────────────┐ │
│          │  │ Chart / Graph         │ │ Recent Activity       │ │
│          │  │                       │ │                       │ │
│          │  └───────────────────────┘ └───────────────────────┘ │
└──────────┴──────────────────────────────────────────────────────┘
```

**Spacing:**
- Page padding: `p-6`
- KPI grid gap: `gap-6`
- Section gap: `gap-8`

## Form Page Layout

Structure for data entry forms.

```
┌──────────────────────────────────────────────────────────────┐
│ [Navbar]                                                     │
├──────────┬───────────────────────────────────────────────────┤
│          │  ← Back                                           │
│          │  Page Header                                      │
│          ├───────────────────────────────────────────────────┤
│ Sidebar  │  ┌─────────────────────────────────────────────┐  │
│          │  │ Card (max-width: 720px)                     │  │
│          │  │                                             │  │
│          │  │ Form fields...                              │  │
│          │  │                                             │  │
│          │  │ [Cancel]                      [Submit]      │  │
│          │  └─────────────────────────────────────────────┘  │
└──────────┴───────────────────────────────────────────────────┘
```

**Guidelines:**
- Form max-width: `max-w-[720px]`
- Field spacing: `space-y-6`
- Button alignment: Right-aligned, primary on right
